# Amazon Sales Data Analysis (Python)

## Project Overview
This project analyzes Amazon sales data to understand order trends, product performance, and shipping patterns.

## Dataset
- ~128,000 records
- 21 features including order status, category, amount, location, and fulfillment details

## Tools & Libraries
- Python
- Pandas, NumPy
- Matplotlib, Seaborn

## What I Did
- Loaded and explored the dataset using Pandas
- Cleaned data by handling missing values and removing irrelevant columns
- Performed exploratory data analysis to identify trends and patterns

## Key Insights
- Analyzed order status and cancellation patterns
- Studied category-wise and location-based sales distribution
- Evaluated fulfillment and shipping behavior

## Outcome
This project demonstrates end-to-end data analysis from raw data to insights using Python.
